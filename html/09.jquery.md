# jQuery 총정리

## 2. 속성(Getter, Setter)

### 내용 선택 및 변경
```js
$('대상').html();       // return 대상의 내용 HTML  Getter
$('대상').html(html);   // return 대상              Setter
$('대상').text();       // return 대상의 내용 text -> tag 안나옴!!
$('대상').text(text);   // return 대상
```

### css
```js
// 아래와 같이 css 속성을 주면 tag에 inline-style로 들어간다.(우선순위가 제일 높아짐!!!)
$('대상').css('opacity');               // return 값
$('대상').css(['opacity', 'left']);     // return 객체(값들) -> 여러개의 속성을 한 번에 가져올 때에는 배열 형태로 쓴다.
$('대상').css('opacity', .5);           // return jQuery 
$('대상').css({
	'opacity': .5,
	'left': '100px',
	});           // return jQuery  -> 여러개의 속성을 한 번에 set 할 때에는 중괄호를 쓴다.

$('대상')addClass('클래스명');
$('대상')removeClass('클래스명');
$('대상')toggleClass('클래스명'); // 이건 뭐지?(검색 필요)
```

### 속성
```js
$('대상').attr('src');                 // return 속성 값
$('대상').attr('src' , 'p1.jpg');      // return 대상

// 단 다음의 속성은 true, false로 준다. (readonly 확인 필요)
$('대상').attr('disabled', true);
```
